Pendataan & Uji Kelayakan Laptop

Tim IT MTani sedang melakukan pendataan kondisi laptop seluruh karyawan untuk menilai kelayakan perangkat terhadap kebutuhan kerja, serta menentukan prioritas pengadaan atau penggantian laptop. Pengisian hanya membutuhkan beberapa menit.

Cara mengisi

  1. 1

    Unduh aplikasi cek spesifikasi

    Pilih aplikasi sesuai sistem operasi laptop Anda melalui tombol di bawah. Aplikasi ini tidak perlu diinstal.

  2. 2

    Jalankan aplikasi

    Untuk Windows, klik dua kali file Check-Laptop.bat. Untuk Mac/Linux, jalankan file bash check-laptop.sh melalui terminal. Spesifikasi laptop akan terdeteksi secara otomatis.

  3. 3

    Browser akan terbuka otomatis

    Setelah aplikasi dijalankan, Anda akan diarahkan ke form pendataan dengan data spesifikasi laptop yang sudah terisi otomatis.

  4. 4

    Lengkapi data dan kirim form

    Lengkapi data diri, informasi aset, dan kondisi laptop, lalu klik Kirim. Skor kelayakan laptop akan langsung ditampilkan setelah form dikirim.
